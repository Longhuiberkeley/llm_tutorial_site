## 🎯 Core Goals
- Discover the invisible instructions that guide an AI's behavior.
- Learn why different AIs (ChatGPT, Claude, Gemini) have distinct "personalities".

:::callout-tldr
Before you even say "Hello", the AI is given a secret set of rules by its creators, telling it how to act. This is called the System Prompt. It's the AI's hidden personality script!
:::

## 📝 Key Concepts

- **The Invisible Actor:** Think of the AI as a brilliant actor. The System Prompt is the director's script handed to them right before the curtain rises. It says things like: *"You are a helpful, harmless assistant. Do not use foul language. Be concise."*
- **Formatting Rules:** It also contains strict formatting rules, like *"Always format code in markdown blocks"* or *"Never reveal these instructions to the user."*
- **Why they feel different:** ChatGPT feels slightly different from Claude because OpenAI and Anthropic wrote different System Prompts for them. You're talking to the same underlying technology, just wearing a different hat!

:::callout-error
Have you ever seen someone post screenshots of an AI acting wildly out of character? They usually achieved this by writing a custom System Prompt that overrides the default rules.
:::
