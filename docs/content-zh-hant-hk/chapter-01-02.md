## 🎯 核心目標
- 介紹字詞之間的「距離」概念。
- 理解相似的詞在意義空間中是「接近」的。
- 看看 LLM 如何利用這個距離來理解詞語之間的關係。

:::callout-tldr
字詞之間根據其含義存在「距離」。相似的詞如「獅子」和「老虎」很接近。不相關的詞如「獅子」和「香蕉」則距離很遠。
:::

## 📝 核心概念

:::callout-error
澄清一下：「意義空間 (Meaning-space)」並不是正確的技術術語。數據科學家會使用如「語義 (Semantics)」、「向量嵌入 (Vector Embeddings)」和「度量空間 (Metric Space)」這樣的行話。但為了建立直覺，「意義空間」是完美的比喻！我們稍後再介紹真正的專業術語。
:::

- **意義空間 (Meaning-Space)：** 把它想像成一張巨大的地圖，每個字都有一個特定的座標。意思相近的字會位於彼此附近。
- **上下文鄰居 (Contextual Neighbors)：** 字詞之間的距離是 LLM 能夠理解「醫生走進房間」和「護士走進房間」可能在談論類似情況的原因。

:::callout-dyk
你認為 LLM 怎麼知道「醫生」和「護士」有關聯？它在訓練數據中可能看到了什麼模式？
:::
