6.3 What Makes a Good Agent?

Responsive and adaptive AI.

🎯 Core Goals

  • Define what makes an AI system "agentic" (vs. just a script).
  • Identify the traits of high-quality agents.
An agent isn't just an AI with tools. It's an AI that can adapt its plan when things go wrong and self-correct until the goal is achieved.

👁️ Visuals & Interactives

The Agent Checklist

What separates a smart script from a true AI agent?

reply

Responsive

Listens to feedback mid-task and adjusts instantly. "Wait, don't search Google, use Bing instead."

published_with_changes

Adaptive

If Tool A fails, the agent tries Tool B or a different search term instead of giving up.

spellcheck

Self-Correcting

Identifies its own mistakes. "I got a math error, let me re-calculate with the calculator tool."

target

Goal-Directed

Stays focused on the final objective, even if the intermediate steps become messy or complex.

The Result
Task Started Problem Solved Autonomously

📝 Key Concepts

  • Responsive: Listens to feedback and changes direction.
  • Adaptive: If a tool fails, it tries a different approach instead of giving up.
  • Self-Correcting: "That output looks wrong, let me double-check my calculation."
  • Goal-Directed: Keeps working toward the final objective, even if the middle steps change.
Contrast: A traditional script follows fixed steps (A -> B -> C). An agent follows a loop: Perceive -> Plan -> Act -> Observe -> Repeat.
