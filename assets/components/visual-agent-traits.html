The Agent Checklist

What separates a smart script from a true AI agent?

reply

Responsive

Listens to feedback mid-task and adjusts instantly. "Wait, don't search Google, use Bing instead."

published_with_changes

Adaptive

If Tool A fails, the agent tries Tool B or a different search term instead of giving up.

spellcheck

Self-Correcting

Identifies its own mistakes. "I got a math error, let me re-calculate with the calculator tool."

target

Goal-Directed

Stays focused on the final objective, even if the intermediate steps become messy or complex.
